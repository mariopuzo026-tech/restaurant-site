# Restaurant Site

## Features
- User-friendly interface for browsing restaurant menus.
- Ability to place online orders with secure payment options.
- Review and rating system for restaurants.
- Search functionality to find restaurants by cuisine or location.
- Responsive design for mobile and desktop users.

## Installation Instructions
1. Clone the repository:
   ```sh
   git clone https://github.com/mariopuzo026-tech/restaurant-site.git
   ```
2. Navigate into the project directory:
   ```sh
   cd restaurant-site
   ```
3. Install the dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
